# NHP agent-registration golden vectors (temporary vendor)

`agent_registration_golden.json` is a **byte-identical temporary vendor** of the
qurl-conformance artifact `qurl-agent-registration-golden-vectors`
(`github.com/layervai/qurl-conformance`, `vectors/agent_registration_golden.json`).

It is copied verbatim (SHA-256-verified against the source) so
`crypto/golden.test.ts` can byte-fence the vendored NHP wire against the
cross-language reference **now**, without waiting on the published npm accessor.

and consumed by `portal.test.ts` via `qv2Vectors()`) does **not yet** expose an
`agentRegistrationVectors()` accessor. Once it does, delete this fixture and load
the vectors through that accessor instead — the same pattern `portal.test.ts`
uses for the qv2/relay-knock vectors.

published npm accessor (the fence itself now exists in-repo; #176 is no longer
